BlueprintCAD.io

Next.js TypeScript React PostgreSQL Vercel Railway Stripe Gemini AI
Problem:

There was no centralized platform for 3D model and CAD file sharing, collaboration, and management. Users needed a way to organize, share, and monetize their 3D assets while providing an interactive viewing experience.

Solution:

Built a full-stack SaaS platform with Next.js and TypeScript (100k+ lines of code) featuring multi-provider authentication (email, Google, GitHub), project management, social collaboration tools, and an interactive 3D viewer. Integrated Stripe for payments, PostgreSQL for data storage on Railway, Cloudflare R2 for asset storage, and Gemini AI for intelligent quote generation. Deployed frontend on Vercel with serverless architecture.

Result:

Created a production-ready platform enabling users to upload, organize, share, and monetize 3D models. The marketplace and collaboration features provide a complete ecosystem for 3D content creators and consumers, with scalable architecture supporting growth.

Architecture: Next.js Frontend (Vercel) ‚Üí API Routes ‚Üí PostgreSQL (Railway) ‚Üí Cloudflare R2 (Assets) ‚Üí Stripe (Payments) ‚Üí Gemini AI (Quotes)

SightGard Twilight - Certification Quiz System

Vue 3 AWS Cognito Lambda DynamoDB API Gateway Healthcare
Problem:

Eye care professionals administering dark adaptation tests needed a certification system to ensure proper training and compliance. The system required randomized quizzes, progress tracking, and certification status management integrated with the clinical management platform.

Solution:

Designed and implemented the certification quiz system using Vue 3 with Composition API. Built a 10-question interactive quiz covering CPT codes, patient selection, test procedures, and device operation. Integrated with AWS Cognito for authentication, Lambda functions for quiz logic, DynamoDB for storing results, and API Gateway for RESTful endpoints. Implemented question randomization, real-time progress tracking, and automatic certification scoring with three-tier results (certified, certified with refresher, or retrain required).

Result:

Provided a comprehensive certification system that ensures operator competency for the dark adaptation test. The quiz history tracking allows clinics to monitor staff certification status over time, improving quality assurance and compliance with clinical standards. The randomized question system prevents memorization and ensures genuine understanding of procedures.

Architecture: Vue 3 Frontend ‚Üí AWS Cognito (Auth) ‚Üí API Gateway ‚Üí Lambda Functions ‚Üí DynamoDB (Quiz Results & History)
